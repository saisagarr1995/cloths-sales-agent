- WhatsApp: @whiskeysockets/baileys (open-source, QR-scan login on a normal WhatsApp number, session persisted in ./auth).

- Payments: UPI deep link upi://pay?pa=<UPI_ID>&pn=<payee>&am=<total>&tn=<orderId> plus a QR PNG generated with the `qrcode` package. Manual confirmation flow (no gateway).
- Persistence: plain JSON files — data/customers/<phone>.json and data/orders.json. No database.

Placeholder solid-colour PNGs generated by a zero-dependency script scripts/gen-images.js using Node's built-in zlib (user replaces with real photos later).
